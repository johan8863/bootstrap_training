Column wrapping

As part of Flexbox features if more than 12 columns are used within a single row, the last column will be placed in a nre line

.col-3
.col-10, 3 + 10 = 13, this column will alway be rendered in a new line.

Offsetting columns

It is possible to offset grid columns in two ways. .offset- grid classes or via margin utilities.

Offset classes

.col-md-4
.col-md-4 .offset-md-4
.col-md-3 .offset-md-3
